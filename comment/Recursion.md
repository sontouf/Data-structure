# Recursion

`재귀(Recursion)`는 문제를 작은 단위로 나누어/정의하여 해결하는 것이다.

## 재귀함수
> 재귀함수는 base case 를 만날 때까지 자기 자신을 호출한다.

#### 1부터 n 까지 합 구하기
```c
int sum(int n) {
    if (n == 1) return 1;
    return sum(n-1) + n
}
```
#### 피보나치 수열
```c
int fibonaci(int n) {
    if (n == 0 || n == 1) return 1;
    return fibonaci(n-2) + fibonaci(n-1);
}
```

## 재귀의 장점과 단점
- 장점
    - 재귀적인 문제를 풀 때 용이하다.
    - 가독성이 반복보다 좋다.

- 단점
    - 함수 호출은 오버헤드를 발생시킨다.
    - 원래의 실행 문맥으로 돌아가기 위해 시스템 스택을 사용하는 데 메모리 사용측면에서 비효율적이다.
    - 반복에 비해 수행 속도가 느리다.
